# DermaScan - تطبيق تشخيص الأمراض الجلدية

تطبيق مكتبي متقدم لتشخيص الأمراض الجلدية باستخدام الذكاء الاصطناعي والتعلم الآلي.

## 🚀 المميزات

- **تشخيص دقيق**: استخدام نموذج ذكي متقدم لتشخيص 7 أنواع من الأمراض الجلدية
- **واجهة عربية**: واجهة مستخدم كاملة باللغة العربية
- **تطبيق مكتبي**: يعمل كتطبيق مستقل بدون الحاجة لمتصفح
- **إدارة متقدمة**: نظام إدارة شامل للمستخدمين والحالات
- **أمان عالي**: نظام تسجيل دخول آمن مع صلاحيات مختلفة

## 🏥 أنواع المستخدمين

### 1. المستخدم العادي (Patient)
- **الوصف**: المستخدم الأساسي للتطبيق
- **الصلاحيات**:
  - إجراء تشخيص للأمراض الجلدية
  - عرض الحالات الخاصة به
  - متابعة نتائج التشخيص
- **التسجيل**: متاح للجميع من صفحة التسجيل

### 2. الطبيب (Doctor)
- **الوصف**: أطباء متخصصون في الأمراض الجلدية
- **الصلاحيات**:
  - جميع صلاحيات المستخدم العادي
  - عرض جميع الحالات في النظام
  - إدارة الحالات والتشخيصات
- **التعيين**: يتم تعيينه من قبل الإدارة فقط

### 3. الإدارة (Admin)
- **الوصف**: مدير النظام المسؤول عن إدارة جميع المستخدمين
- **الصلاحيات**:
  - جميع صلاحيات الطبيب
  - إدارة المستخدمين (إضافة، تعديل، حذف)
  - ترقية المستخدمين العاديين إلى أطباء
  - إدارة جميع الحالات

## 🎯 كيفية الاستخدام

### للمستخدمين الجدد
1. **إنشاء حساب**: انتقل إلى صفحة "إنشاء حساب جديد"
2. **إدخال البيانات**: أدخل الاسم والبريد الإلكتروني وكلمة المرور
3. **تأكيد التسجيل**: سيتم إنشاء حسابك كـ مستخدم عادي
4. **تسجيل الدخول**: استخدم بياناتك للدخول إلى النظام

### للإدارة
1. **تسجيل الدخول**: استخدم أحد الحسابات الإدارية الافتراضية
2. **إدارة الأطباء**: انتقل إلى "إدارة الأطباء" من القائمة
3. **ترقية المستخدمين**: اختر مستخدم عادي وارفع صلاحياته إلى طبيب
4. **إدارة الحالات**: يمكنك عرض وحذف جميع الحالات في النظام

### للأطباء
1. **تسجيل الدخول**: استخدم حسابك المعتمد من الإدارة
2. **عرض الحالات**: يمكنك رؤية جميع الحالات في النظام
3. **إدارة التشخيصات**: مراجعة وتحديث التشخيصات

## 🏥 الأمراض المدعومة

النظام يدعم تشخيص 7 أنواع من الأمراض الجلدية:

1. **التقرن الشعاعي** (Actinic keratoses)
2. **سرطان الخلايا القاعدية** (Basal cell carcinoma)
3. **آفات شبيهة بالتقرن الحميد** (Benign keratosis-like lesions)
4. **الورم الليفي الجلدي** (Dermatofibroma)
5. **الشامات الميلانينية** (Melanocytic nevi)
6. **سرطان الجلد الميلانيني** (Melanoma)
7. **آفات الأوعية الدموية** (Vascular lesions)
